Course Content

1.

Course Introduction
6 Sessions

This section will include all the introductory course materials to help students without any prior experience in web development of the language syntax and semantics.

  • Session 1: - Introduction,
    - How the web works,
    - Roles of HTML and CSS in Web Development,
    - Setup our development environment.
  • Session 2: - Create a Github account,
    - Learn GIT basics - Why HTML is exciting,
    - First look at HTML (structure),
    - Hands on, create our first HTML page.
  • Session 3: - HTML document structure - doctype and basic layout,
    - Linking HTML pages together,
    - Semantic structural elements,
    - Non-semantic elements (div & span),
    - Section element.
  • Session 4: - Navigation,
    - Text basics: adding headings, lists, special characters, formatting (bold and italics),
    - Setting up a live server,
    - Meta tags and search engines.
  • Session 5: - Paragraphs and Typography,
    - Tables,
    - Forms, inputs, and choosing between options.
  • Session 6: - Block and in-line level elements,
    - Classes and Ids,
    - Adding media to a web page - images, audio files, video files and vector graphics.

2.

CSS Fundamentals
6 Sessions

In this section, we are introduced to CSS which is used to style and layout web pages. You can alter the font, colour, size and spacing of your content, split it into multiple columns, or add animations and other decorative features.

  • Session 7: - Introduction to CSS,
    - CSS syntax,
    - Single and multiple stylesheets,
    - Implementing CSS,
    - Inline and External CSS.
  • Session 8: - CSS selectors in detail,
    - ID Selectors,
    - Class Selectors,
    - Group Selectors,
    - Universal Selectors,
    - Descendant / Child Selectors,
    - CSS Pseudo Classes.
  • Session 9: - Fonts in CSS,
    - Backgrounds, borders and outline,
    - Box model, margin and padding, height and width, dimesions.
  • Session 10: - Float and Text alignment,
    - Styling the Navigation menu,
    - Links and button styling.
  • Session 11: - Inline, Block and inline-block display,
    - Positioning (Relative, Static, Absolute and Fixed),
    - Visibility, Order and Negative space.
  • Session 12: - Overlapping content and Transparent backgrounds,
    - Full-width content, Fixed-width content,
    - Use developer tools or Inspect element browser feature.

3.

Intermediate CSS
7 Sessions

In this section, we continue to explore more interesting properties of CSS, by applying it more in styling elements to enhance their appearance.

  • Session 13: - CSS Color,
    - Box shadows,
    - Rounded corners.
  • Session 14: - Styling Forms,
    - Styling Data tables (border, width and height, text-align, vertical align, padding, color).
  • Session 15: - CSS Transform,
    - CSS Transitions.
  • Session 16: - CSS Animations,
    - CSS Scrollbars.
  • Session 17: - CSS lists - list-style-type, list-style-position, list-style-image,
    - CSS Floats - The float property, Clear property, Clearfix hack.
  • Session 18: - Responsive web design,
    - Responsive Grids.
  • Session 19: - Introduction to Flexbox,
    - Responsive grid with Flexbox.

Web Design
5 Sessions

In this section, we should focus on effective web design which is majorly determined by the website users. It should always be easy to use.

  • Session 20: - Introduction to web design,

  • Session 21: - Working with Images,
    - Working with Icons,
    - Spacing and Layouts.
  • Session 22: - The Grid system (CSS Grid),

  • Session 23: - Responsive design with Media Queries,
    - EM and REM Units,
    - Vh and Vw Units.
  • Session 24: - Getting started with Bootstrap 4.5,
