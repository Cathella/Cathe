Course Content

1.

Course Introduction
4 Lessons

This section will include all the introductory course materials to help students without any prior experience in web development of the language syntax and semantics.

  • Lesson 1: - Introduction,
    - How the web works,
    - Roles of HTML and CSS in Web Development,
    - Setup our development environment,
    - Create a Github account,
    - Learn GIT basics
  • Lesson 2: - Why HTML is exciting,
    - First look at HTML (structure),
    - Hands on, create our first HTML page,
    - HTML document structure - doctype and basic layout,
    - Linking HTML pages together,
    - Semantic structural elements.
  • Lesson 3: - Navigation,
    - Non-semantic elements (div & span),
    - Section element,
    - Text basics: adding headings, lists, special characters, formatting (bold and italics),
    - Setting up a live server,
    - Meta tags and search engines.
  • Lesson 4: - Paragraphs and Typography,
    - Tables,
    - Forms, inputs, and choosing between options,
    - Block and in-line level elements,
    - Classes and Ids,
    - Adding media to a web page - images, audio files, video files and vector graphics.

2.

CSS Fundamentals
5 Lessons

In this section, we are introduced to CSS which is used to style and layout web pages. You can alter the font, colour, size and spacing of your content, split it into multiple columns, or add animations and other decorative features.

  • Lesson 5: - Introduction to CSS,
    - CSS syntax,
    - Single and multiple stylesheets,
    - Implementing CSS,
    - Inline and External CSS.
  • Lesson 6: - CSS selectors in detail,
    - ID Selectors,
    - Class Selectors,
    - Group Selectors,
    - Universal Selectors,
    - Descendant / Child Selectors,
    - CSS Pseudo Classes.
  • Lesson 7: - Fonts in CSS,
    - Backgrounds, borders and outline,
    - Box model, margin and padding, height and width, dimesions
    - Float and Text alignment.
  • Lesson 8: - Styling the Navigation menu,
    - Links and button styling,
    - Inline, Block and inline-block display,
    - Positioning (Relative, Static, Absolute and Fixed).
  • Lesson 9: - Visibility, Order and Negative space,
    - Overlapping content and Transparent backgrounds,
    - Use developer tools or Inspect element browser feature,
    - Full-width content, Fixed-width content.

Intermediate CSS

  • Lesson 10: - Box shadows,

  • Lesson 11: - CSS Color,
    - CSS Transform,
    - CSS Transitions,

  • Lesson 12: - CSS Scrollbars,

  • Lesson 13: - Responsive web design,
    - Responsive Grids,
    - Responsive grid with Flexbox,
    - Introduction to Flexbox.

Web Design
