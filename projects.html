Project: Homeschool

Skills:

Ruby Ruby on Rails Google Maps APIs Stimulus Bootstrap

Homeschool matches teachers with students in a study-group in the same location. It uses Google Maps API to obtain addresses of teachers and students. Students that are near each other are grouped in study-groups that are then matched with a teacher near them.
